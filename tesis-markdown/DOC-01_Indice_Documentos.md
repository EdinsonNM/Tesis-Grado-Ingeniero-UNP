3.  Realidad problemática (DOC-06) → Formulación (DOC-07) → Justificación (DOC-08)

4.  Objetivos (DOC-09) → Delimitación (DOC-10) → Bases teóricas (DOC-11)

5.  Glosario (DOC-12) → Hipótesis (DOC-13) → Matrices (DOC-14 y DOC-15)
