# Variable dependiente

## Definicion

Variable cuyo comportamiento se mide para observar el efecto de una intervencion experimental.

## En la tesis

La variable dependiente principal es el nivel de degradacion de respuestas, operacionalizado mediante el [[IQCR]].

## Indicadores asociados

- [[Exactitud de respuesta]]
- [[Tasa de alucinacion]]
- [[Coherencia conversacional]]
- [[Eficiencia de tokens]]

## Relacion experimental

Las tecnicas de memoria y gestion de contexto son la variable independiente. El IQCR permite observar si dichas tecnicas producen una mejora medible.
