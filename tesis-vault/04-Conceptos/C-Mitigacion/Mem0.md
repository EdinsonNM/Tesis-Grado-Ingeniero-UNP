# Mem0

## Definicion

Sistema de memoria adaptativa para agentes LLM que distingue entre memorias de usuario, agente y sesion, combinando recuperacion vectorial y estructuras tipo grafo.

## Valor para la tesis

Sirve como referencia conceptual para memoria longitudinal categorizada y para evitar mezclar preferencias estables con estado transitorio.
