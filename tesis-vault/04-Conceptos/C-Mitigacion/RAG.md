# RAG

## Definicion

Retrieval-Augmented Generation. Tecnica que recupera documentos o fragmentos relevantes desde una memoria externa y los entrega al modelo como contexto para generar una respuesta.

## En la tesis

RAG se considera una tecnica individual de memoria externa para reducir la necesidad de mantener todo el historial conversacional en el contexto activo.

## Variantes relevantes

- RAG sobre historial de conversacion.
- [[RAG-MCP]] sobre catalogos de herramientas.
- Recuperacion hibrida densa y lexica.
- Recuperacion jerarquica.
